# Welcome to Kilt

**Kilt** (Kernel-based Inverse Laplace Transform) is a Python package for 
fast and flexible fluorescence lifetime analysis using the 
**Maximum Entropy Method (MEM)** with support for **1D** and **2D global fitting**.
